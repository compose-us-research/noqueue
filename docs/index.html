Platzhalter.io - ein Werkzeug fürs einfache Abstandhalten

Das Problem

Wenn die Geschäfte, Gaststätten und Studios wieder öffnen, müssen sie irgendwie ihre Besucher, Gäste und Patienten managen, so dass die Abstände etc. eingehalten werden können.

Die Gäste haben ebenfalls ein Interesse daran sich nicht in vollen Geschäften, Praxen etc. aufzuhalten.

Dieser Umstand wird bis mindestens Frühjahr 2021 so bleiben.

Die Lösung

Mit "Platzhalter.io" können Gewerbetreibende Ihre Verfügbarkeiten einfach managen und so Ihren Auflagen nachkommen.

Der Kunde wiederum bucht sein Vorhaben entspannt per Smartphone. So werden Menschenschlangen vor und in Geschäften im Vorhinein vermieden.

"Platzhalter.io" vereint dabei einfache Bedienbarkeit mit Datensparsamkeit, welche wiederum das Geschäfts -und Zusammenleben trotz Covid19 erlaubt.

Wenn "Platzhalter.io" eine Erleichterung ist, dann kann der Nutzer spenden. Die Vermarktung passiert über die Wirtschaftsförderungen, Social Media und Website an die betroffenen Betreiber.

Die Technik

Gewerbetreibende können ihren Kunden feste Slots und Termine zuweisen und so Schlangen, Wartezeiten und Abstandsprobleme einfach umgehen. Es ist kein Technik-KnowHow, kein Installieren nötig - das gilt sowohl für die Gewerbetreibenden als auch für den Kunden.

Wie oben schon erwähnt, ist uns Datensparsamkeit wichtig - es werden keine Kundendaten erhoben, außer die Einrichtung bspw. der Frisör muss gesetzlichen Auflagen nachkommen. In diesem Fall wird der Nutzer darauf hingewiesen.

Das Ganze ist natürlich Open Source, das heißt der Code steht der Gemeinschaft zur Verfügung - zur Einsicht und zur Weiterentwicklung.
